# VPC Peering Lab

This project demonstrates creating a **VPC peering connection** between two VPCs in AWS, including routing, VPC Flow Logs, testing connectivity, and analyzing traffic.

## Lab Objectives
- Create a VPC peering connection
- Configure route tables to use the peering connection
- Enable VPC Flow Logs
- Test connectivity between EC2 and database
- Analyze network traffic
