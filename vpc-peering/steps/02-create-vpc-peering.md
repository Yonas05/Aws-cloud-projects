# Step 2 — Create VPC Peering Connection

1. In the VPC Console → **Peering Connections → Create Peering Connection**
2. Configure:
   - **Name tag:** Lab-Peer
   - **Requester VPC:** Lab VPC
   - **Accepter VPC:** Shared VPC
3. Click **Create Peering Connection**
4. Select the newly created connection → **Actions → Accept Request**
5. Verify the status changes to **Active**
