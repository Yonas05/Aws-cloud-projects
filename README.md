# AWS VPC Labs

Welcome to the **AWS VPC Labs** repository! This repo provides hands-on labs to learn and practice creating secure, scalable networking environments in AWS. You will work with **VPCs, subnets, NAT gateways, bastion hosts, VPC peering, security groups, and network ACLs**.

These labs are ideal for students, cloud enthusiasts, and professionals preparing for AWS certifications.

## Lab Objectives

- Learn to create **public and private subnets**.
- Launch and configure **EC2 instances** securely.
- Implement **NAT gateways** and enable internet access for private resources.
- Configure **VPC peering** and routing between VPCs.
- Use **VPC Flow Logs** to analyze network traffic.
- Apply **security best practices** using security groups and custom NACLs.
